-----------------------------------------------------------------------
Thing Specifications
-----------------------------------------------------------------------
Thing's format is very similar to Hexen's with the exception of specials

* ---------Thing Format---------

    * short x;
    * short y;
    * short z;
    * short angle;
    * short type;
    * short options;
    * short tid;


* ---------Thing Flags---------

    * Easy
    * Medium
    * Hard
    * Deaf
    * Multiplayer
    * Spawn
    * Trigger on Pickup   - Triggers linedefs when picked up
    * Trigger on Death    - Triggers linedefs when killed
    * Disable Gravity??   - Unknown
    * Disable Infighting  - Monsters will not change targets unless is a player
