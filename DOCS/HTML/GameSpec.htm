GAME SPECIFICATIONS
-----------------------------------------------------------------------
This section is a rough summary that covers all existing Doom mechanics that were modified or removed in Doom64

* ---------General---------

    * P_Move: actor->z = actor->floorz has been removed which enemies fall gradually to the floor when going down stairs.
    * F_Finale: Sprites for each 'cast' can be rotated
    * Redesigned Intermission screen/text
    * Automap no longer has markers, and grid. Zooming/panning is much faster
    * Player arrow in automap is replaced with a pulsating triangle.
    * Secret exits no longer hardcoded. Uses linetag of line special 124 to indicate secret map.
    * HU_Stuff: Simplified. Less widgets.
    * HU Text fades out instead of dissapearing
    * HU Text is limited to one text message on screen. EX changed the limit back to 4 messages (max) on screen.
    * Menu system redesigned. Fade in/out effects
    * Slower clock rate, which the game runs a lot slower than PC Doom. (EX retains the same speed as the PC game).
    * Simplified player HUD system. Lots of code for it has been removed.
    * Flashing key cards when attempting to open lock door without a key. (JaguarDoom code).
    * Unpeg Top Linedef flag also aligns the y offset of all textures to the nearest 64 grid

* ---------Player---------

    * Taller view height
    * Wider radius and height
    * Weapon sprites centers on the screen when firing
    * Weapon flash code pointers (A_Light0, A_Light1, etc) no longer used
    * Only one death sfx
    * Random hud messages appears (mock messages) when dead for a certain amount of time
    * Unable to wall run (EX retains this bug/feature).
    * Hud flashes (damage, radioactive suit, invulnerable, etc) replaced with 3D overlay.
    * View pitch recoil from certain weapons
    * Super Shotgun, and Rocket Launcher pushes player back
    * Green flash when firing BFG
    * Health and Armor pickups boosts points by 2 instead of 1 (Jaguardoom feature).
    * Chaingun sprite shakes when firing
    * Shake screen effects
    * Hud message when finding secrets
    * Plasma gun contains animation and sound loops
    * Pistol, Chainsaw, Chaingun, Shotgun, Super Shotgun and Plasma Rifle has faster firing rates.
    * Faster weapon scrolling (N64 only. EX is attempting to emulate this feature).

* ---------Thing---------

    * Alpha values (transclucent)
    * New spawning mechanics
    * Different projectiles for Hell Knight and Baron
    * No Revenant, Arch-Vile, Spider Mastermind or Chaingun Zombie
    * Shotgun Zombie moves faster than Pistol Zombie
    * Specture has new fading effects
    * Arachnotron's firing rate/behavior has been modified. Fires two projectiles
    * Lost Souls more aggressive
    * Pain Elemental fires two Lost Souls instead of one
    * Imps now has melee frames
    * NOGRAVITY flags has been changed to GRAVITY behavior. Things without this flag now floats.
    * MF_SLIDE replaced with new flag that activates linedef specials upon death.
    * Cyberdemon explodes during death
    * RESURRECTOR boss
    * More blood spawned when thing is shot
    * Cacodemons more aggressive and stronger.
    * Cyberdemon's rocket is more faster. Sprite is aligned to the Cyberdemon's cannon.
    * Items can trigger linedef specials on pickup.

* ---------Level Interaction---------

    * Ceiling/Floor movement speed has been increased
    * Crusher height from floor has been increased (to prevent clipping into player's view z).
    * Non-Switch doors moves the floor sector as well as the ceiling. (Split Doors).
    * Redesigned linedef activation system
    * Completly customizable linedef triggering
    * Switches redesigned. Linedef creates a subline which is used to render the switch
    * Switch textures are no longer 'merged' with the actual texture, but instead are reserved in a unused sidedef texture (top, middle, lower).
    * Macro scripting system
