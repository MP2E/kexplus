Tutorials
-----------------------------------------------------------------------
This article will roughly explain some of the level design tutorials for Doom64.
As before, this article will assume that you are already familiar with Doom mapping.

* ---------Split Door ---------

    * Create door sector
    * Raise ceiling by 64 units, and raise floor by 64 units
    * Texture lower sidedef
    * Assign any manual door linedef special (1, 117, 118 etc)
    * To keep the door tracks from moving, assign both Unpeg Top and Unpeg Lower flags

* ---------Thing Spawning---------

    * Add a thing
    * Assign a TID. (For this example, assign 1 as the TID).
    * Set 'Spawn' flag on thing.
        * Note that when this flag is set, the thing will not be present in the level.
    * Setup a linedef to spawn the thing.
    * Set special to 224 and set the tag to 1 (Thing TID).
    * When triggered, the thing will spawn in. Spawning will work on all things in the game.

    * Fake Item Pickups
       
        Fake Items can be found under editor ID: 89. Fake items are invisible objects that the
        the player can pickup. No flash, or message appears when this item is picked up. This
        is very useful for creating scripted events.

 * ---------Death Triggering---------  

    * Assign a TID to the enemy or enemies that need to trigger something when killed.
    * Set the 'Trigger On Death' flag.
    * Setup a linedef to trigger the special.
        * Set the flag 'Activated By Dead Thing'
        * Make sure the tag is the same as the thing TID
        * Make sure that the line cannot be activated by the player, or make the line unreachable
          to the player.
    * Note that if there are multiple enemies with the same TID, the linedef will not activate
      until all enemies with the same TID are dead.

* ---------See-Through Ceilings/Floors---------

    In Doom64, its possible to use masked textures for the ceiling and floors, giving the room
    a unique 3D appearence.

    * Create a room that is 128 units tall and 384x384 units wide.
    * Create another sector inside that is 128x128 units wide.
    * Move the sector's ceiling height up by 64 units.
    * Texture the ceiling flat with a sky texture. (Note: Only skies can create this effect).
    * Within that sector, create another sector.
        * Make sure the boundaries of the new sector is very close to the borders of the other
          sector.
        * Move the ceiling down by 32 units.
        * Texture the ceiling with a masked texture. (Grate, fence etc).

* ---------Creating Dart/Tracer Launchers---------

    * Add thing type 2050 (Projectile Spot)
    * Assign a TID
    * For launching Darts, use line special 202 and for Tracers, use line special 231
    * Line tag will be the TID of the projectile spot that the dart or tracer should launch from
